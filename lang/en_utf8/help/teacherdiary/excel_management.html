Teacher diary: add teacher diary pages using Excel

Teacher diary module allow you to add multiple pages uploading a CSV file that must follow this schema (the first row must be the heading):

Name Description
id ID number for this page. Leave it blank if you want to add a new page
name The name of the teacher diary. It must be the the same name you find as label in the course (it is case sensitive)
course ID number of the course
teachername The name of the teacher
day day of the lecture (between 1 and 31)
month month of the lecture (between 1 and 12)
year year of the lecture
starthour starting hour of the lecture (between 1 and 24)
startminute starting minute of the lecture (between 0 and 49)
endhour final hour of the lecture (between 1 and 24)
endminute final minute of the lecture (between 0 and 49)
summary summary about the topic of the lecture

This structure is called editable Excel format and you can download it from the page "Add teacher diary pages using Excel" for each teacher diary. You can you this file to copy-paste the pages you already have inserted to go faster to create your new pages.
