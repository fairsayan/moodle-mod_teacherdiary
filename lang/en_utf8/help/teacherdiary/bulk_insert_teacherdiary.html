Teacher diary: how to add multiple teacher diaries

You can create many teacher diaries together uploading a CSV file. The first row must contain the names of all the fields, in this table you have the meaning of each field:

Name Description
teacher Username of the teacher who will be the owner of the diary
course ID course number
section The number of the section in which must be inserted the diary
name Name of the diary
plannedhours Planned hours for this diary

If already exists another diary with the same name in a course, the new diary won't be created.
